MATHS: IMPORTANT FORMULAS AND SOLUTION TRICKS

🔥 Integration by Parts — Power Reducing Quick Formulas

💡 Type 1: ∫ xⁿ × eˣ dx

Quick Formula:

  • ∫ xⁿ × eˣ dx = xⁿ × eˣ - n ∫ xⁿ⁻¹ × eˣ dx

💥 Repeat till x power becomes 0 — fast!

💡 Type 2: ∫ xⁿ × sin(x) dx or ∫ xⁿ × cos(x) dx

Quick Formula:

  • ∫ xⁿ × sin(x) dx = -xⁿ × cos(x) + n ∫ xⁿ⁻¹ × cos(x) dx
  • ∫ xⁿ × cos(x) dx = xⁿ × sin(x) - n ∫ xⁿ⁻¹ × sin(x) dx

💥 Repeat till x power becomes 0 — easy peasy.

🚀 Example: Fast Application

  • ∫ x × eˣ dx = x × eˣ - ∫ eˣ dx = x × eˣ - eˣ + C ✔️
  • ∫ x² × eˣ dx = x² × eˣ - 2 ∫ x × eˣ dx = x² × eˣ - 2(x × eˣ - eˣ) + C ✔️
  • ∫ x × sin(x) dx = -x × cos(x) + ∫ cos(x) dx = -x × cos(x) + sin(x) + C ✔️

💥 MCQ Killer Hack:
Memorize these formulas.
Spot the power dropping pattern.
For x × eˣ or x² × sin(x) — use formula and reduce the power step by step.
Options usually match after 1 or 2 reductions — no full by-parts drama needed!

⚡ More By Parts Quick Formulas for Instant Solving

💡 Type 3: ∫ xⁿ × ln(x) dx

Quick Formula:

  • ∫ xⁿ × ln(x) dx = (xⁿ⁺¹ / (n + 1)) × ln(x) - ∫ (xⁿ / (n + 1)) dx

💥 Only ONE by-parts needed — done!

💡 Type 4: ∫ xⁿ × sin(ax) dx or ∫ xⁿ × cos(ax) dx

Quick Formula:

  • ∫ xⁿ × sin(ax) dx = -xⁿ × cos(ax)/a + n/a ∫ xⁿ⁻¹ × cos(ax) dx
  • ∫ xⁿ × cos(ax) dx = xⁿ × sin(ax)/a - n/a ∫ xⁿ⁻¹ × sin(ax) dx

💥 Same as sin(x) type, just divide by 'a' for ax inside!

💡 Type 5: ∫ xⁿ × e^(ax) dx

Quick Formula:

  • ∫ xⁿ × e^(ax) dx = (xⁿ × e^(ax))/a - (n/a) ∫ xⁿ⁻¹ × e^(ax) dx

💥 Power drops by 1 each time — keep applying till done!

🚀 Example Applications

  • ∫ x × ln(x) dx = (x²/2) × ln(x) - x²/4 + C ✔️
  • ∫ x² × e^(3x) dx = (x² × e^(3x))/3 - (2/3) ∫ x × e^(3x) dx ✔️
  • ∫ x × sin(2x) dx = -x × cos(2x)/2 + (1/2) ∫ cos(2x) dx = -x × cos(2x)/2 + sin(2x)/4 + C ✔️

💥 Turbo MCQ Trick:
Spot 'xⁿ' × function? Check:
✅ Trig? Use sin/cos formulas.
✅ Exponential? Use e^(ax) pattern.
✅ ln(x)? Direct formula, 1 step done.
No full by-parts drama — just reduce power and apply pattern till x vanishes!

🔥 Bonus By Parts Quick Formulas — MCQ Power-Up

💡 Type 6: ∫ ln(x) dx

Shortcut:

∫ ln(x) dx = x × ln(x) - x + C

💥 Classic by-parts type — direct formula, no steps needed!

💡 Type 7: ∫ tan⁻¹(x) dx

Shortcut:

∫ tan⁻¹(x) dx = x × tan⁻¹(x) - (1/2) × ln(1 + x²) + C

💥 If you spot inverse trig alone — this formula saves time instantly.

💡 Type 8: ∫ sin⁻¹(x) dx

Shortcut:

∫ sin⁻¹(x) dx = x × sin⁻¹(x) + √(1 - x²) + C

💥 Again — classic inverse trig — direct shortcut, MCQ done!

💡 Type 9: ∫ eˣ × sin(x) dx

Shortcut:

∫ eˣ × sin(x) dx = (eˣ × (sin(x) - cos(x))) / 2 + C

💥 This is called a "repeat integral" — direct shortcut avoids full steps.

💡 Type 10: ∫ eˣ × cos(x) dx

Shortcut:

∫ eˣ × cos(x) dx = (eˣ × (sin(x) + cos(x))) / 2 + C

💥 Similar to previous — pure MCQ-time saviour.

💥 High-Yield Tip:
If you memorize these formulas, 80% of by-parts MCQs in competitive exams are auto-solved. These types come again and again — they love asking them!

📦 Common Integrals — Trig & Inverse Trig Functions

🎯 Basic Trigonometric Integrals

  • ∫ sin(x) dx = -cos(x) + C
  • ∫ cos(x) dx = sin(x) + C
  • ∫ sec²(x) dx = tan(x) + C
  • ∫ csc²(x) dx = -cot(x) + C
  • ∫ sec(x) tan(x) dx = sec(x) + C
  • ∫ csc(x) cot(x) dx = -csc(x) + C

🎯 Inverse Trigonometric Integrals

  • ∫ sin⁻¹(x) dx = x sin⁻¹(x) + √(1 - x²) + C
  • ∫ cos⁻¹(x) dx = x cos⁻¹(x) - √(1 - x²) + C
  • ∫ tan⁻¹(x) dx = x tan⁻¹(x) - (1/2) ln(1 + x²) + C
  • ∫ cot⁻¹(x) dx = x cot⁻¹(x) + (1/2) ln(1 + x²) + C
  • ∫ sec⁻¹(x) dx = x sec⁻¹(x) - ln|x + √(x² - 1)| + C
  • ∫ csc⁻¹(x) dx = x csc⁻¹(x) + ln|x + √(x² - 1)| + C

🎯 Other Useful Trig Integrals

  • ∫ sin²(x) dx = (x/2) - (sin(2x)/4) + C
  • ∫ cos²(x) dx = (x/2) + (sin(2x)/4) + C
  • ∫ sin(x) cos(x) dx = (sin²(x))/2 + C

💡 Quick MCQ Trick:
If you see inverse trig in an integral — think of ready-made by-parts formulas. For basic trig like sin²(x) or cos²(x), apply identity first before integrating!

🎯 Extra Common Trig & Inverse Trig Integrals

🔥 More Trigonometric Integrals

  • ∫ tan(x) dx = -ln|cos(x)| + C
  • ∫ cot(x) dx = ln|sin(x)| + C
  • ∫ sec(x) dx = ln|sec(x) + tan(x)| + C
  • ∫ csc(x) dx = ln|csc(x) - cot(x)| + C
  • ∫ sin²(x) dx = (x/2) - (sin(2x)/4) + C
  • ∫ cos²(x) dx = (x/2) + (sin(2x)/4) + C

🔥 More Inverse Trigonometric Integrals

  • ∫ dx / √(1 - x²) = sin⁻¹(x) + C
  • ∫ dx / (1 + x²) = tan⁻¹(x) + C
  • ∫ dx / (|x| √(x² - 1)) = sec⁻¹(x) + C

🔥 Power of Trig Integrals (Quick IDs)

  • ∫ sin³(x) dx — Break as sin²(x) × sin(x), use identity sin²(x) = 1 - cos²(x)
  • ∫ cos³(x) dx — Similar: cos²(x) × cos(x), use identity cos²(x) = 1 - sin²(x)
  • ∫ sinᵐ(x) cosⁿ(x) dx — Reduce using identities if either m or n is odd

💡 Quick MCQ Rule:
If denominator looks like √(1 - x²), think sin⁻¹(x);
If it's (1 + x²), think tan⁻¹(x);
If it's √(x² - 1), think sec⁻¹(x). These show up everywhere in exams!

⭕ Circles — Must-Know Formulas & Fast Tricks

⭕ Circle Maths - Types of Questions, Solving Steps & Examples

1. Find Centre & Radius

General Equation: x² + y² + 2gx + 2fy + c = 0

Steps:

  • Centre = (-g, -f)
  • Radius = √(g² + f² - c)

Example: Find centre & radius of x² + y² - 6x + 8y - 11 = 0

Solution:

  • 2g = -6 → g = -3
  • 2f = 8 → f = 4
  • c = -11
  • Centre = (3, -4)
  • Radius = √(3² + (-4)² - (-11)) = √(9 + 16 + 11) = √36 = 6

2. Find Equation of Circle with Given Centre & Radius

Standard Form: (x - h)² + (y - k)² = r²

Example: Find equation of circle with centre (2, -1) and radius 5

Solution: (x - 2)² + (y + 1)² = 25

3. Find Length of Tangent from a Point

Formula: Length = √(x₁² + y₁² + 2g x₁ + 2f y₁ + c)

Example: Find tangent length from (1, 2) to x² + y² - 4x + 6y - 3 = 0

Solution:

  • 2g = -4 → g = -2
  • 2f = 6 → f = 3
  • c = -3
  • Length = √(1² + 2² + 2(-2)(1) + 2(3)(2) - 3) = √(1 + 4 - 4 + 12 - 3) = √10

4. Find Equation of Tangent at a Point on Circle

Formula: xx₁ + yy₁ + g(x + x₁) + f(y + y₁) + c = 0

Example: Tangent at (2, 1) to x² + y² - 4x + 2y + 1 = 0

Solution:

  • 2g = -4 → g = -2
  • 2f = 2 → f = 1
  • c = 1
  • Equation: x(2) + y(1) -2(x + 2) + 1(y + 1) + 1 = 0 → 2x + y -2x -4 + y +1 +1 = 0 → 2y -2 = 0 → y = 1

5. Condition for Tangent from a Line

Step: Perpendicular distance from centre to line = radius

Example: Find k if line 3x - 4y + k = 0 is tangent to x² + y² = 25

Solution:

  • Centre = (0, 0), Radius = 5
  • Distance = |k| / √(3² + (-4)²) = 5
  • |k| / 5 = 5 → |k| = 25 → k = ±25

6. Two Circles Touch Externally or Internally

Formulas:

  • Touch externally: Distance between centres = Sum of radii
  • Touch internally: Distance = Difference of radii

Example: Circles with centres (0, 0) & (6, 0), radii 2 & 4 — find touch type

Solution: Distance = 6, Sum = 2 + 4 = 6 → Touch externally ✅

📐 Conic Sections - Quick Guide, Types, Formulas & Examples

1. Parabola

Standard Equations:

  • y² = 4ax → Opens right
  • y² = -4ax → Opens left
  • x² = 4ay → Opens upward
  • x² = -4ay → Opens downward

Key Points:

  • Vertex: (0, 0)
  • Focus: (a, 0) or (0, a) based on orientation
  • Directrix: x = -a or y = -a

Example: Find focus of y² = 12x

Solution: 4a = 12 → a = 3 → Focus = (3, 0)

2. Ellipse

Standard Equations:

  • (x² / a²) + (y² / b²) = 1 → Horizontal major axis (a > b)
  • (x² / b²) + (y² / a²) = 1 → Vertical major axis (a > b)

Key Points:

  • Centre: (0, 0)
  • Foci: (±c, 0) or (0, ±c) where c² = a² - b²
  • Length of Major Axis: 2a
  • Length of Minor Axis: 2b

Example: Find foci for x²/25 + y²/9 = 1

Solution: a = 5, b = 3 → c² = 25 - 9 = 16 → c = 4 → Foci = (±4, 0)

3. Hyperbola

Standard Equations:

  • (x² / a²) - (y² / b²) = 1 → Opens left & right
  • (y² / a²) - (x² / b²) = 1 → Opens up & down

Key Points:

  • Centre: (0, 0)
  • Foci: (±c, 0) or (0, ±c) where c² = a² + b²
  • Asymptotes: y = (±b/a)x or y = (±a/b)x

Example: Find foci for x²/16 - y²/9 = 1

Solution: a = 4, b = 3 → c² = 16 + 9 = 25 → c = 5 → Foci = (±5, 0)

📘 Conic Sections - Extra Solved Questions with Full Steps

Parabola - Find Vertex, Focus & Directrix

Question: For the parabola x² = -16y, find vertex, focus, and directrix.

Solution:

  • Equation: x² = -4ay → opens downward
  • 4a = 16 → a = 4
  • Vertex: (0, 0)
  • Focus: (0, -4)
  • Directrix: y = 4

Parabola - Find Equation with Given Vertex & Focus

Question: Find equation of parabola with vertex at (0, 0) and focus at (2, 0).

Solution:

  • Opens right (focus on positive x-axis)
  • a = 2 → Equation: y² = 4ax → y² = 8x

Ellipse - Find Foci & Eccentricity

Question: For the ellipse x²/25 + y²/9 = 1, find foci & eccentricity.

Solution:

  • a = 5, b = 3
  • c² = a² - b² = 25 - 9 = 16 → c = 4
  • Foci: (±4, 0)
  • Eccentricity: e = c/a = 4/5 = 0.8

Ellipse - Find Lengths of Axes

Question: Find lengths of major & minor axes for (x²/36) + (y²/16) = 1.

Solution:

  • a² = 36 → a = 6 (major axis)
  • b² = 16 → b = 4 (minor axis)
  • Length of Major Axis = 2a = 12
  • Length of Minor Axis = 2b = 8

Hyperbola - Find Foci & Asymptotes

Question: Find foci & asymptotes of hyperbola x²/9 - y²/16 = 1.

Solution:

  • a² = 9 → a = 3
  • b² = 16 → b = 4
  • c² = a² + b² = 9 + 16 = 25 → c = 5
  • Foci: (±5, 0)
  • Asymptotes: y = ±(b/a)x = ±(4/3)x

Hyperbola - Find Eccentricity

Question: Find eccentricity for hyperbola y²/25 - x²/16 = 1.

Solution:

  • a² = 25 → a = 5
  • b² = 16 → b = 4
  • c² = a² + b² = 25 + 16 = 41 → c = √41
  • Eccentricity: e = c/a = √41 / 5 ≈ 1.28
